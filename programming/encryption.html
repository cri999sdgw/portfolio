Encryption

Exercise

Encrypt a given text with the Caesar Encryption, programmed in the language C!

Program Description

Method in which each letter in the plaintext is replaced by a letter some fixed number of positions down the alphabet. The method is named after Julius Caesar, who used it in his private correspondence. The key by which the characters get shifted is calculated via 256 mod #InputCharLength.

User Input

The user has to do input the text to encode.

Restrictions

The usage of external libraries was prohibited (expect the standard libraries). To make it run, I used three different libraries:
- stdio.h
- stdlib.h

Purpose

The whole purpose of the program was, to pass a subject for my Degree in Computer Science. the subject was named "Programming 1", where I made other small projects like Hangman or the the game Solitair, which are also shown on the website. The main Goal of the Subject was, to teach the fundamentals of the programming language C.
